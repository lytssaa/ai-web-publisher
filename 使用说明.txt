# AI 网页一键发布工具

HTML/ZIP 网页一键上线,生成专属访问链接。

## 快速开始

### 本地使用(仅自己电脑能访问)

双击 `一键启动.bat`,自动打开浏览器。

- 首页: http://localhost:8000/index.php
- 后台: http://localhost:8000/admin.php  (默认密码 admin123)

停止服务:关闭命令行窗口即可。

### 公网使用(任何人都能访问)

双击 `一键启动_公网版.bat`,等待几秒后会显示一个 `https://*.trycloudflare.com` 的地址,这就是你的公网网址。

**前提**:需要安装 cloudflared:
```
winget install Cloudflare.cloudflared
```

公网模式下,生成的 `SITE_URL` 需要手动修改为 cloudflared 显示的地址,否则生成的链接会是 localhost,别人打不开。

修改方法:编辑 `config.php`,把第一行 `SITE_URL` 改成显示的公网地址(结尾带斜杠)。

## 目录结构

```
网页转链接/
├── php/              PHP 运行环境(已配置好,无需额外安装)
├── uploads/          生成的网页文件(自动创建)
├── data/             数据库(自动创建)
├── index.php         首页(上传/生成网页)
├── site.php          网页访问入口
├── admin.php         管理后台
├── config.php        配置文件
├── functions.php     函数库
├── db.php            数据库初始化
├── 一键启动.bat      本地启动
└── 一键启动_公网版.bat  公网启动
```

## 后台功能

- 数据概览:站点数、卡密数等
- 卡密管理:生成/删除卡密
- 卡密规格:天卡/月卡/年卡等
- 站点管理:查看/删除已生成的网页
- 系统设置:开启免费模式、修改密码、设置有效期等

## 使用说明

### 上传方式

1. **上传文件**:支持 .html / .htm / .zip(最大 10MB,ZIP 根目录需含 index.html)
2. **粘贴代码**:直接粘贴 HTML 代码,支持实时预览

### 链接自定义

- 「想要的链接后缀」可自定义,如填 `my-page`,访问地址为 `/uploads/my-page/`
- 不填则自动生成 16 位随机 ID

### 短链接

勾选「同时生成短链接」可生成短链,便于分享。

## 换电脑部署

1. 把整个文件夹拷贝到新电脑
2. 双击 `一键启动.bat`
3. 完事

无需安装 PHP、无需配置环境,开箱即用。

## 部署到服务器(长期固定域名)

1. 购买虚拟主机/云服务器(支持 PHP + SQLite)
2. 上传所有文件(除了 php/ 文件夹,服务器自己有 PHP)
3. 修改 `config.php` 中的 `SITE_URL` 为你的域名
4. 设置 `uploads/` 和 `data/` 目录可写权限
5. 访问域名即可使用

## 默认密码

后台默认密码:admin123

**首次使用请立即到后台 → 系统设置 修改密码!**

## 注意事项

- 免费模式下生成的网页有有效期,到期自动删除
- 禁止上传违法、钓鱼、诈骗、侵权内容
- 上传文件有白名单限制,只允许静态资源(html/css/js/图片/字体/音视频)
- 公网模式下的 trycloudflare.com 域名是临时的,每次重启会变,仅用于测试
